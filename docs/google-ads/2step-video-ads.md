# Google Ads -- 2Step Video Reviews Campaign

## Budget Recommendation

Do NOT split from the A$200/month auditandfix.com budget. The A$200 is already tight for 3 search campaigns. Run this as a separate budget when ready.

**Recommended separate budget:** A$150/month minimum (A$5/day)
